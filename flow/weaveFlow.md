Weave (Presentation Generator)

User visits /weave page —> Selects preferred language from 22+ Indian languages and presentation style (Academic, Business, Storytelling, Technical) —> System configures AI model for selected language and presentation format —> User enters presentation topic and selects target audience (School, College, Professional, Training) —> User chooses presentation template from available designs with cultural relevance options —> User selects number of slides (3-20 slides based on content complexity) —> AI suggests slide structure with recommended headings based on topic —> User customizes individual slide headings and content requirements —> System processes input using DeepSeek R1 via Groq API for fast inference —> AI generates complete presentation slides with appropriate content depth for selected audience —> Generated slides include relevant images, diagrams, and infographics fetched from media repository —> (Optional) 3D visualizations added for STEM/technical topics where applicable —> Presentation displayed in slide preview mode with navigation controls —> User can edit individual slides, reorder slides, or regenerate specific slides —> AI provides alternative content suggestions for better engagement —> Speaker notes automatically generated for each slide to assist presenters —> Export options available in multiple formats (PDF, PPTX, Google Slides) —> Presentation can be saved to user account for future editing and sharing —> Offline functionality allows access to previously generated presentations without internet
