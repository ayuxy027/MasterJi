LMR (Learning Material Resource)

User visits LMR page —> Selects preferred language and AI model from multiple options (DeepSeek R1, Groq Llama, Gemini Pro, Claude 3) —> Uploads study materials (PDF, PPT, DOCX formats supported) —> System processes documents: PDFs are chunked via LangGraph with metadata embedded using Embedding Gemma —> Embeddings stored in and retrieved from Chroma vector database —> Selected AI model (with DeepSeek R1 as primary) processes content by retrieving relevant embeddings from Chroma —> Generates outputs in selected language including: —> 1) Content summary —> 2) Key questions —> 3) Quiz/practice tests —> 4) Last-minute recall notes —> 5) Neatly formatted downloadable PDF —> User can solve generated quizzes and analyze performance based on uploaded documents
