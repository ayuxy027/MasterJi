Image Generation (Posters)

User visits posters page —> Selects preferred language and AI image model from multiple options (LLaVA, LoRA, Qwen VL, etc.) —> Chooses educational category (Science, Math, History, etc.) and sets image quality parameters —> Enters prompt for educational poster generation —> System applies cultural context parameters based on selected language/region —> AI model generates 3 distinct but relevant educational posters ensuring variety in visual representation —> All 3 posters are contextually relevant to the input query with Indian cultural relevance —> Generated posters displayed in responsive 3-grid layout for comparison —> User can select best poster, download individual posters or all at once —> Quality settings allow customization for different device capabilities and use cases
