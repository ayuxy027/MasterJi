## Design Principles

### Consistency Rules
- All icons should use the same stroke width (2px)
- Consistent use of strokeLinecap="round" and strokeLinejoin="round"
- Same viewBox dimensions (0 0 24 24)
- Simple geometric shapes that represent the concept clearly
- Minimalist approach focusing on core meaning

### Color Scheme
- Primary: `text-orange-500` (as used in current navbar)
- Hover state: `text-orange-600`
- Can be adjusted based on context but maintain consistent styling

### Sizing
- Standard: `w-5 h-5` (20px by 20px)
- Can be adjusted as needed while maintaining aspect ratio
